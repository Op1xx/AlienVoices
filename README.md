# 👽 AlienVoices

Программа для классификации радиосигналов инопланетных цивилизаций.  
Проект для МПОШ 2026, сектор "ИТ".

---

## 📌 О проекте

Учёный Михаил получает сигналы от инопланетян, но их слишком много для ручной обработки.  
Нужно:
- Восстановить повреждённые метки классов в данных
- Обучить нейросеть для классификации
- Сделать приложение с графикой и авторизацией

---

## 🏗 Как работает

```
Пользователь → Интерфейс (PyQt6) → Сервер (Flask) → Нейросеть → Результат
```

**Нейросеть:**
- Входной слой — принимает аудио (WAV)
- Скрытые слои — ищут паттерны
- Выходной слой — показывает вероятность класса

---

## 🛠 Технологии

- **ML:** TensorFlow, Librosa, NumPy
- **Backend:** Flask
- **GUI:** PyQt6, Matplotlib
- **БД:** NumPy (.npz)
- **Тесты:** pytest
